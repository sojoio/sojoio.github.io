CORVID Cyber Security

Email Security Management App

Web app · User & Project Research · Userflow · Wireframes · Prototypes · AdobeXD · HTML · SCSS

Project Overview

The goal of this project was to overhaul the current user UI and create and additional Admin UI allowing clients take over some areas of configuration and management of their email security system. This included the addition of many new security features and much improved data visualisation. The current UI nolonger conformed to the standards I had incorrperated in to new products. The development of a new interface for this service meant I would be working towards my own wider goal of achieving unification and familiarty for users accross all of our services applications.

My Contributions

I was the sole designer on this project taking on all elements of UI/UX. The current application was to be built again from scrach with new technology and frameworks so I knew I had opprotuty to really improve the user experience of this application. I undertook all project research (including user and competitor), hat to consider user flow for 4 auth levels, wireframing and prototyping. I also took on the majority of the front end development of this application with support from team members back end work.
